# My AI Secretary – Demedia Web Application

Ce projet est une **application web front‑end** présentant le concept **My AI Secretary**, un assistant intelligent conçu pour aider les utilisateurs à parcourir et comprendre rapidement de longs documents.

---

## Présentation du projet
- Application web **statique avec intégration PHP**
- Interface simple et moderne
- Page d’accueil mettant en avant le concept d’assistant IA

---

## Fonctionnalités principales
### Page d’accueil
- Titre principal **My AI Secretary**
- Slogan : *Asking is faster than skimming*
- Bouton **GET STARTED** menant à la page d’inscription

### Navigation
- Barre de navigation intégrée via **PHP (`include`)**
- Structure modulaire du site

---

## Structure du projet
- `index.html` / `index.php` : page d’accueil
- `nav_bar.php` : barre de navigation réutilisable
- `style.css` : styles du site
- `inscription.html` : page d’inscription utilisateur

---

## Technologies utilisées
- **HTML5**
- **CSS3**
- **PHP**
- Responsive design

---

## Objectif pédagogique
Ce projet permet de :
- Comprendre la structure d’une **application web simple**
- Utiliser **PHP pour modulariser le code**
- Créer une **landing page claire et efficace**
- Mettre en valeur un **concept d’application IA**

---

## Lancement du projet
- Placer le projet dans un dossier serveur (ex : `htdocs`)
- Lancer un serveur local (XAMPP, WAMP, MAMP)
- Accéder au projet via `http://localhost/`
